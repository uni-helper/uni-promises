`uni-app` promise 化的 API。

## 起步

### 安装依赖

安装依赖。

使用 `npm`：

```shell
npm install @uni-helper/uni-promises
```

使用 `yarn`：

```shell
yarn install @uni-helper/uni-promises
```

使用 `pnpm`：

```shell
pnpm install @uni-helper/uni-promises
```

使用 `cnpm`：

```shell
cnpm install @uni-helper/uni-promises
```
